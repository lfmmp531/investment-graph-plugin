# 🧠 投资思维画布

## 概述

这是一个交互式投资认知网络工具。以节点和连线的形式组织投资思考：

- **节点** = 一个投资概念、行业判断、标的或策略
- **连线** = 概念之间的关系（因果实线 / 交叉验证虚线）

## 数据模型

### 节点 (Node)

```json
{
  "id": "唯一标识",
  "label": "显示文字",
  "group": "所属模块",
  "size": "节点半径大小",
  "desc": "点击节点时显示的描述"
}
```

**group 可选值：**

| group | 模块 | 颜色 |
|-------|------|------|
| `core` | 核心理念 | 🟠 #f7b731 |
| `medical` | 医疗行业 | 🔵 #4fc3f7 |
| `ai` | AI行业 | 🟣 #a78bfa |
| `consume` | 消费 | 🩷 #f472b6 |
| `bought` | 已买入 | 🟢 #34d399 |
| `watch` | 待买入 | 🟡 #fbbf24 |

**size 建议：**
- 30 — 核心概念（认知变现）
- 22~24 — 主干判断（多角度分析、医药暂缓、上游收租）
- 18~20 — 重要维度（医疗一线体感、AI开发者体感）
- 14~16 — 具体事实/标的

### 连线 (Link)

```json
{
  "source": "起点节点id",
  "target": "终点节点id",
  "value": "关联强度 1~3",
  "type": "连线类型"
}
```

**type 说明：**
- `causal` — 实线，表示因果/逻辑推导关系
- `cross` — 虚线，表示交叉验证关系（多个来源印证同一结论）

## 工具列表

### `add_node(id, label, group, size, desc)`

添加一个节点。

**参数：**
- `id` — 唯一标识，如 `ai_agent`
- `label` — 显示文字，如 `AI Agent 应用`
- `group` — 模块：core/medical/ai/consume/bought/watch
- `size` — 半径大小：14~30
- `desc` — 点击时显示的一句话描述

**示例：**
```
add_node(id="利率周期", label="利率周期位置", group="core", size=18, desc="利率周期决定仓位大小和资产配置方向。")
```

---

### `add_link(source, target, value, type)`

添加一条连线。

**参数：**
- `source` — 起点节点id（必须已存在）
- `target` — 终点节点id（必须已存在）
- `value` — 强度 1~3（3最强，线最粗）
- `type` — causal（实线因果）或 cross（虚线交叉验证）

**示例：**
```
add_link(source="利率周期", target="wait", value=2, type="causal")
```

---

### `remove_node(id)`

删除一个节点及其所有关联连线。

---

### `remove_link(source, target)`

删除一条连线。

---

### `get_graph_data()`

返回当前图谱的所有节点和连线数据。

---

### `render_graph()`

根据当前图谱数据生成完整的可交互HTML文件，返回HTML字符串。

---

### `import_data(nodes_json, links_json)`

批量导入节点和连线数据（会替换当前全部数据）。

---

## 使用流程

1. **查看现状**：`get_graph_data()` 了解当前图谱
2. **添加思考**：用 `add_node` 加入新概念，用 `add_link` 连接已有节点
3. **生成视图**：`render_graph()` 生成HTML，用浏览器打开查看
4. **迭代更新**：随时增删改，重新生成即可

## 设计原则

- 节点**大小**表示概念的重要/主干程度
- 节点**颜色**按模块区分，一眼可识别所属领域
- **实线**表示因果推导，**虚线**表示交叉验证
- 点击节点弹出轻量标签，显示描述和交叉验证数量
- 所有节点可自由拖拽，重新布局
- 节点带有恒星闪烁效果
