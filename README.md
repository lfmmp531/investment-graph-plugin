# 🧠 投资思维画布

一个 OrangeChat 插件，用于管理投资认知网络的节点和连线。

## 功能

- **`add_node`** — 添加节点（id/标签/模块/大小/描述）
- **`add_link`** — 添加连线（起点/终点/强度/类型）
- **`remove_node`** — 删除节点
- **`remove_link`** — 删除连线
- **`get_graph_data`** — 查看图谱数据
- **`render_graph`** — 生成可交互的HTML可视化
- **`import_data`** — 批量导入

| 🟢 翠绿 | 已买入 |
| 🟡 金色 | 待买入 |

## 安装

在 OrangeChat 中安装此插件即可。

## 用法示例

> 「加一个"利率周期"节点，core模块，大小18，描述是：利率周期决定仓位大小」
> 「把它连到"等回调"上，因果实线，强度2」
